# transformer
This repository presents HyperGAE-Transformer, a novel deep learning framework for breast cancer survival prediction using heterogeneous multi-modal data obtained from The Cancer Genome Atlas. The proposed framework integrates six complementary modalities, including Clinical data, Copy Number Variation (CNV), DNA Methylation, miRNA expression, mRNA expression, and Whole-Slide Imaging (WSI) features, to capture diverse biological characteristics associated with breast cancer prognosis. After preprocessing and patient-wise alignment across all modalities, a total of 1035 breast cancer patients were included in this study. Patients were categorized into short-term and long-term survivors based on a five-year survival threshold to enable effective prognosis modeling and clinical risk assessment. The HyperGAE-Transformer framework combines modality-specific hypergraph construction, Hypergraph Autoencoder-based representation learning, and Transformer-based attention fusion to model higher-order biological interactions and long-range cross-modal dependencies across heterogeneous modalities. By integrating multi-omics and imaging information within a unified framework, the proposed model aims to improve the robustness and accuracy of breast cancer survival prediction for precision oncology applications
